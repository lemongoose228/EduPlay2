Архитектура системы

Трёхуровневая модель: SPA, NestJS (модули из кода), Redis для TTL-таймеров сессий, PostgreSQL через TypeORM.

Стрелки показывают основные потоки: HTTP и WebSocket от клиента; сессии используют Redis и БД; остальные модули обращаются к данным через TypeORM.
